Aperçu

Avez-vous besoin de vous rafraîchir la mémoire concernant la syntaxe de Markdown? Excellent!

Vous trouverez ci-dessous la liste des éléments les plus courants et utilisés de la langue.

Il est à noter que, ici seules les bases de Markdown sont présentées, si vous avez besoin de plus d'informations, consultez les sections de syntaxe de base et syntaxe étendue.

Syntaxe de base

Les éléments qui sous-tendent Markdown sont indiqués ci-dessous. Ces éléments de formatage ont été décrits dans le document en langue d'origine et la grande majorité des programmes les prennent en charge.

Élément Markdown
Titres 
# H1
## H2
### H3
Essayez-le vous-même

**texte en gras**

*texte en italique*
Essayez-le vous-même 
Blockquote 
> Blockquote
Essayez-le vous-même 
Liste ordonnée 
1. Premier élément
2. Deuxième élément
3. Troisième élément
Essayez-le vous-même 
Liste non ordonnée 
- Premier élément
- Deuxième élément
- Troisième élément
Essayez-le vous-même

`code`
Essayez-le vous-même 
Ligne horizontale

Essayez-le vous-même 
Link 
[Titre](https://markdown.net.br)

Image

Syntaxe étendue

Avec l'évolution de Markdown, d'autres éléments ont été implémentés, enrichissant la gamme d'options de mise en forme des documents. D'une part, un peu plus de complexité, mais d'autre part, la langue a gagné plus de ressources pour la construction de différents documents.

Gardez à l'esprit que toutes les applications développées pour Markdown ne supportent pas ces éléments, c'est toujours une bonne idée de consulter la documentation du programme ou la section "outils".
