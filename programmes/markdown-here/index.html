Markdown Here

Markdown Here est une extension pour Browser qui vous permet de convertir le contenu Markdown saisi dans un champ de formulaire sur le site Web au format Rich Text.

  1. Markdown
  3. Éditeurs Markdown

Aperçu

L'extension Markdown Here donne à l'utilisateur la possibilité d'emporter Markdown partout où il va.

À tout le moins, nous pouvons dire que l'outil est assez ambitieux, car le nombre de sites différents est astronomique et l'application est aventureuse dans l'ensemble.

Dans tout? Je veux dire, les sites qui prennent en charge le formatage Rich Text, oui, ce type de formatage Microsoft Word tel que gras, italique et souligné.

Des sites comme Gmail, Hotmail et WordPress sont des exemples d'endroits où Markdown Here peut être utilisé.

Mais ne vous arrêtez pas là, Markdown Here fonctionne même dans la version Web de Evernote.

En cliquant ici, vous pouvez consulter une liste de services compatibles avec l'outil.

Pour utiliser Markdown Here, installez d'abord l'extension pour votre navigateur, visitez un site Web comme Gmail, tapez le contenu de votre email en utilisant le formatage Markdown, cliquez sur le bouton opposé de mouse (bouton droit) dans le formulaire de message et cliquez sur Markdown Toggle.

Voir l'image ci-dessous.

Là, Markdown Here fera le travail de convertir votre contenu Markdown en Rich Text dans le site lui-même.

Bien sûr, en raison du nombre absurde de sites offrant un formatage différent, il peut y avoir des incohérences dans le contenu rendu.

Vérifiez toujours la mise en forme de votre message, assurez-vous qu'il correspond à ce que vous voulez avant de l'envoyer à quelqu'un.

Le site officiel de Markdown Here se trouve à https://markdown-here.com/.
