Standard Notes

Standard Notes est une application de prise de notes simple, sécurisée et open source.

Aperçu

Standard Notes est une application de prise de notes gratuite qui offre d'excellentes fonctionnalités à l'utilisateur.

L'outil ne prend pas en charge Markdown par défaut, mais vous pouvez l'activer lors de l'achat de la licence étendue pour l'application.

Avec cette licence vous avez accès à plusieurs extensions pour modifier le comportement du programme, l'une d'elles est Advanced Markdown.

Les éléments Markdown répertoriés dans cet article sont basés sur cette extension.

Mais pourquoi utiliser Standard Notes?

Avec une grande variété d'options parmi les applications Markdown, pourquoi utiliser Standard Notes?

Eh bien, il offre à l'utilisateur une excellente expérience avec la synchronisation, la confidentialité et plusieurs extensions.

L'équipe Standard Notes est également engagée dans la philosophie open source, donnant au programme plus de transparence et augmentant la fiabilité du projet.

La compatibilité est une autre force de app, la plupart des plates-formes ont des versions disponibles pour elles.

Vous pouvez trouver plus d'informations sur Standard Notes sur le site officiel https://standardnotes.org/.
