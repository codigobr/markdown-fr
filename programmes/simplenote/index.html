Simplenote

Simplenote est un programme d'annotation simple compatible avec Markdown.

  1. Markdown
  3. Éditeurs Markdown

Aperçu

L'application d'annotation Simplenote a été développée par la société Automattic, créateur de WordPress.

Simplenote a des versions pour Web, exécutées directement dans votre navigateur, ainsi que des versions pour appareils mobiles et Desktop.

La compatibilité est également excellente, le programme est disponible pour la plupart des systèmes d'exploitation, y compris Windows, macOS, Linux et Android.

Simplenote est gratuit et open source, vous pouvez visiter le référentiel d'outils à GitHub et avoir accès au code pour toutes les versions.

Pour utiliser le programme, il est nécessaire de créer un compte, avec cela, vos documents seront synchronisés sur tous vos appareils via le serveur Automattic.

Il est important de savoir que Automattic ne prend pas en charge le cryptage de vos fichiers et que l'option de synchronisation ne peut pas être désactivée.

Simplenote a une option d'exportation intéressante appelée Publish to Web, dans laquelle l'utilisateur peut générer un URL public pour un document et le partager avec d'autres.

Activation de la prise en charge de Markdown sur Simplenote

Vous pouvez activer le support Markdown dans Simplenote en cliquant sur l'icône d'information et en sélectionnant l'option: Markdown Formatted.

Voir l'exemple dans l'image ci-dessous:

Lorsque vous activez cette option, elle devient la valeur par défaut pour les annotations suivantes, c'est-à-dire qu'à partir de maintenant vous pouvez utiliser Markdown.

Pour plus d'informations sur Simplenote, vous pouvez visiter le site officiel de l'outil à https://simplenote.com/.

Comment prévisualiser les documents Markdown sur Simplenote

Simplenote n'a pas d'éditeur en temps réel, c'est-à-dire que les formats Markdown ne sont pas affichés pendant que l'utilisateur tape.

Pour visualiser le rendu d'un document Markdown dans Simplenote, cliquez sur l'icône en forme d'œil dans le coin supérieur droit.
